Algorithms Final Project

Ned Cramer, Brian Ha, Avery Gu, Andrew Chough

A project designed to draw a recursive tree for any algorithm.

Select an Algorithm:
Input:

          LIS: Input NUMBERS, COMMA DELIMITED.
          LPS: Input ANY list of characters.
          QuickSort: Input ANY list of characters.
          Fibonacci: Input a NUMBER less than 10.
          Sum First N Integers: Input an INTEGER > 0.
          Factorial: Input an INTEGER > 0.
          Sum of Integer List: Input NUMBERS, COMMA DELIMITED.
